## Artifacts là gì?

Artifacts là khung làm việc trực quan bên cạnh cuộc trò chuyện. AI có thể tạo và hiển thị nội dung tương tác: biểu đồ, hình ảnh, code, và tài liệu.

## 4 loại Artifact

<CardGroup cols={2}>
  <Card title="Interactive Image" icon="image">
    Hình ảnh SVG tương tác — biểu đồ, infographic, minh họa
  </Card>
  <Card title="Generative Diagram" icon="diagram-project">
    Sơ đồ tự động — flowchart, mindmap, architecture diagram
  </Card>
  <Card title="Content Visualizer" icon="chart-mixed">
    Trực quan hóa dữ liệu — bảng, biểu đồ, dashboard
  </Card>
  <Card title="AI Rendering" icon="code">
    Code chạy được — React components, HTML, interactive apps
  </Card>
</CardGroup>

## Cách sử dụng

<Steps>
  <Step title="Yêu cầu AI tạo artifact">
    Mô tả nội dung bạn muốn. Ví dụ:
    - "Tạo biểu đồ so sánh 3 phương pháp nghiên cứu"
    - "Vẽ flowchart quy trình PRISMA"
    - "Tạo dashboard từ dữ liệu: [dữ liệu]"
  </Step>
  <Step title="Xem artifact">
    Artifact hiển thị trong panel bên phải. Bạn có thể tương tác, zoom, và khám phá.
  </Step>
  <Step title="Chỉnh sửa">
    Nói với AI những thay đổi bạn muốn: "Thay đổi màu biểu đồ", "Thêm cột thứ 4", "Làm chữ to hơn".
  </Step>
  <Step title="Xuất">
    Nhấn nút **Download** hoặc **Copy** trên artifact để lưu hoặc chia sẻ.
  </Step>
</Steps>
