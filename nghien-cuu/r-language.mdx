## Tổng quan

Pho.Chat hỗ trợ viết và chạy code R trực tiếp trong cuộc trò chuyện. Bạn có thể yêu cầu AI viết code phân tích thống kê, tạo biểu đồ, và giải thích kết quả.

## Cách sử dụng

<Steps>
  <Step title="Yêu cầu AI viết code R">
    Mô tả phân tích bạn cần. Ví dụ: "Viết code R để chạy meta-analysis từ dữ liệu sau: [dữ liệu]"
  </Step>
  <Step title="Xem và chỉnh sửa code">
    AI sẽ tạo code R trong Artifact. Bạn có thể xem, chỉnh sửa, và chạy.
  </Step>
  <Step title="Xem kết quả">
    Kết quả phân tích và biểu đồ hiển thị trực tiếp trong chat.
  </Step>
</Steps>
